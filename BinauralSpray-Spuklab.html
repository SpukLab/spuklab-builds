Dispersión

// 02 — Workflow
Cómo usar BinauralSpray
01
Cargá tu audio

Importá cualquier archivo de audio desde tu biblioteca o conectá el micrófono en vivo. El buffer circular captura hasta 5 segundos para granularización en tiempo real.

02
Posicioná en el espacio

El eje horizontal es el campo estéreo completo. Mové el toque para desplazar la fuente granular entre el canal L (cyan) y el canal R (magenta).

03
Modulá en tiempo real

Ajustá los 5 parámetros mientras tocás. Densidad construye la textura. Glitch la rompe. Feedback crea bucles orbitales. Todo simultáneo.

04
Grabá el output

Capturá el resultado directamente desde la app en formato WAV. O integrá via AUv3 en tu host favorito — AUM, GarageBand, Cubasis, Logic Pro.

// 05 — Especificaciones
Arquitectura técnica
// Motor Granular
Fragmentación de audio en micropartículas por segundo. 6 parámetros: Chop Density, Binaural Dispersion, Glitch Probability, Feedback, Particle Velocity y Pitch. Control total sobre densidad, velocidad, distribución temporal y tono global.
// Espacialización Binaural
HRTF binaural via Web Audio PannerNode. Cadena Dual Dry/Wet — señal directa y capa granular procesadas en paralelo. El eje táctil horizontal modula ambas capas simultáneamente en tiempo real.
// Pitch Global
Knob PITCH controla el playbackRate global sobre el dry source y los granos. Centrado en 0, rango ±2 octavas. Cambia el tono del material sin afectar la arquitectura granular.
// Signal Protection
Feedback remapeado internamente — el knob muestra 0–100% pero el gain real escala con hard cap para evitar inestabilidad. Brick-wall compressor (DynamicsCompressorNode) + WaveShaper safety net. Clipping indicator en header. El output nunca clipea.
// Library & Grabación
Library sheet con todos los takes grabados — nombre, duración y fecha. Cuatro acciones por take: preview directo, cargar como fuente de audio, exportar WAV vía share sheet iOS, o borrar. Los takes se guardan en IndexedDB (base BinauralSprayLib) y sobreviven cierres de Safari. El flujo REC ya no descarga automático — el botón Library flashea cyan al terminar.
// Skin & Plataforma
Skin canónica "Frío Violeta" — bg linear-gradient(155deg, #13111f, #1b1726). Knobs glassmorphism con arcos tornasolados cyan→violeta→magenta. Tipografía Inter + JetBrains Mono italic. Optimizado para iPad iOS en Safari.
